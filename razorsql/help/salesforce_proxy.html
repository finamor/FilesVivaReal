Salesforce Proxy Configuration

For users that connect to the internet via a proxy server, such as users behind a corporate firewall that only allow internet connections via a proxy, proxy information is required in order for RazorSQL to be able to communicate with Salesforce. Below is more information on the fields that can be configured.

Proxy Host: The host name of the proxy server

Proxy Port: The port of the proxy server

Proxy User: The user name used to authenticate the connection to the proxy server

Proxy Password: The password for the proxy user
