RazorSQL JDBC Bridge URL

This is the URL or address of the RazorSQL JDBC Bridge.

For example, if the RazorSQL JDBC Bridge is deployed on a secure server called www.acme.com, and the name of the RazorSQL JDBC Bridge JSP page is razorsql_jdbc_bridge.jsp and is in the razorsql directory, the URL would be the following:

https://www.acme.com/razorsql/razorsql_jdbc_bridge.jsp

For more information, see the RazorSQL JDBC Bridge User's Guide in the help documentation.
