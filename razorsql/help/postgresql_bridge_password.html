RazorSQL PostgreSQL Bridge Password

To prevent unauthorized access to the RazorSQL PostgreSQL Bridge, a password can be configured in the RazorSQL PostgreSQL Bridge PHP page. If a password is set, the RazorSQL PostgreSQL Bridge will only execute if the passed in password matches the configured password.

The default password for the RazorSQL PostgreSQL PHP Bridge is radmin.

For more information, see the RazorSQL PostgreSQL Bridge User's Guide in the help documentation.
