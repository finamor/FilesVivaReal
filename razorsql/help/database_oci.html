Database

For Oracle OCI connections, the database field should contain one of the following formats:

1. TNSName: - If your machine is setup to resolve TNSNames, this format can be used. Simply enter the name of the database.

2. Host, port, and Service. Enter the data in the following format:

server_name_or_ip_address:port/service_name

3. Host, port, and SID. Enter the data in the following format:

servername_or_ip_address:port:sid
