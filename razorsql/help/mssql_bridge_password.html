RazorSQL MS SQL Server Bridge Password

To prevent unauthorized access to the RazorSQL MS SQL Server Bridge, a password can be configured in the RazorSQL MS SQL Server Bridge PHP or ASP page. If a password is set, the RazorSQL MS SQL Server Bridge will only execute if the passed in password matches the configured password.

The default password for the RazorSQL MS SQL Server PHP or ASP Bridge is radmin.

For more information, see the RazorSQL MS SQL Server Bridge User's Guide in the help documentation.
