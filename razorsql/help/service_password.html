RazorSQL JDBC Bridge Password

To prevent unauthorized access to the RazorSQL JDBC Bridge, a password can be configured for the RazorSQL JDBC Bridge. If the password is set in the JSP page, the RazorSQL JDBC Bridge will only execute if the passed in password matches the configured password.

The default password for the RazorSQL JDBC Bridge is radmin.

For more information, see the RazorSQL JDBC Bridge User's Guide in the help documentation.
