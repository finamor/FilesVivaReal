RazorSQL MS SQL Server Bridge URL

This is the URL or address of the RazorSQL SQL Server PHP or ASP Bridge.

For example, if the RazorSQL MS SQL Server PHP or ASP Bridge is deployed on a secure web server called www.acme.com, and the name of the RazorSQL MS SQL Server Bridge PHP or ASP page is razorsql_sqlserver_bridge.php or razorsql_sqlserver_bridge.asp and is in the razorsql directory, the URL would be the following:

https://www.acme.com/razorsql/razorsql_sqlserver_bridge.php
or
https://www.acme.com/razorsql/razorsql_sqlserver_bridge.asp

The RazorSQL MS SQL Server Bridge PHP or ASP page can be found in the bridge directory under the RazorSQL installation directory.

For more information, see the RazorSQL MS SQL Server Bridge User's Guide in the help documentation.
