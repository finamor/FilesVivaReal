AWS Server

The Amazon Web Services (AWS) server endpoint that contains the domains you would like to work with.

The following are examples of server endpoints. The first set listed is for SimpleDB. The second set is for DynamoDB:

SimpleDB Endpoints

sdb.amazonaws.com (Endpoint located in the US-East Region)

sdb.us-west-1.amazonaws.com (Endpoint located in the US-West Region)

sdb.eu-west-1.amazonaws.com (Endpoint located in the EU Region)

sdb.ap-southeast-1.amazonaws.com (Endpoint located in the Asia Pacific Region)

DynamoDB Endpoints

dynamodb.us-east-1.amazonaws.com (Endpoint located in the US-East Region)

dynamodb.us-west-1.amazonaws.com (Endpoint located in the US West (Northern California) Region)

dynamodb.us-west-2.amazonaws.com (Endpoint located in the US West (Oregon) Region)

dynamodb.eu-west-1.amazonaws.com (Endpoint located in the EU (Ireland) Region)

dynamodb.ap-southeast-1.amazonaws.com (Endpoint located in the Asia Pacific (Singapore) Region)

dynamodb.ap-southeast-2.amazonaws.com (Endpoint located in the Asia Pacific (Sydney) Region)

dynamodb.sa-east-1.amazonaws.com (Endpoint located in the South America (Sao Paulo) Region)
