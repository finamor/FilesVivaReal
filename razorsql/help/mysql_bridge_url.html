RazorSQL MySQL Bridge URL

This is the URL or address of the RazorSQL MySQL PHP Bridge.

For example, if the RazorSQL MySQL PHP Bridge is deployed on a secure web server called www.acme.com, and the name of the RazorSQL MySQL Bridge PHP page is razorsql_mysql_bridge.php and is in the razorsql directory, the URL would be the following:

https://www.acme.com/razorsql/razorsql_mysql_bridge.php

The RazorSQL MySQL Bridge PHP page can be found in the bridge directory under the RazorSQL installation directory.

For more information, see the RazorSQL MySQL Bridge User's Guide in the help documentation.
