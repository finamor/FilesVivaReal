RazorSQL PostgreSQL Bridge URL

This is the URL or address of the RazorSQL PostgreSQL PHP Bridge.

For example, if the RazorSQL PostgreSQL PHP Bridge is deployed on a secure web server called www.acme.com, and the name of the RazorSQL PostgreSQL Bridge PHP page is razorsql_postgresql_bridge.php and is in the razorsql directory, the URL would be the following:

https://www.acme.com/razorsql/razorsql_postgresql_bridge.php

The RazorSQL PostgreSQL Bridge PHP page can be found in the bridge directory under the RazorSQL installation directory.

For more information, see the RazorSQL PostgreSQL Bridge User's Guide in the help documentation.
