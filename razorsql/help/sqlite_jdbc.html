SQLite Connection Help

RazorSQL versions 4.0.1 and greater ship with built-in support for SQLite verison 3.

RazorSQL communicates with SQLite via the RazorSQL SQLite Driver, which communicates to SQLite via the SQLite command line program. The SQLite command line tool ships with RazorSQL for Windows, and is pre-bundled with Mac OS X version 10.4 and is located here: /usr/bin/sqlite3. For other operating systems, the SQLite command line tool will have to be manually installed.

JDBC drivers are also available for SQLite. However, it is recommended that users use the RazorSQL SQLite Driver if possible.
