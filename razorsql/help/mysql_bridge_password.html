RazorSQL MySQL Bridge Password

To prevent unauthorized access to the RazorSQL MySQL Bridge, a password can be configured in the RazorSQL MySQL Bridge PHP page. If a password is set, the RazorSQL MySQL Bridge will only execute if the passed in password matches the configured password.

The default password for the RazorSQL MySQL PHP Bridge is radmin.

For more information, see the RazorSQL MySQL Bridge User's Guide in the help documentation.
