Product Description RazorSQL is a unversal database query tool, SQL editor, and database navigator. It includes a relational database engine that requires no configuration. Some of the main features contained in RazorSQL are visual tools for creating, editing, describing, altering, dropping, and viewing tables; tools for importing and exporting data; a database browser for the viewing of database objects and structures; and a robust programming editor with support for SQL, PL/SQL, TransactSQL, SQL PL, Java, XML and many other programming languages.
